# MCP Agent 前端應用

這是一個使用React開發的前端應用，用於與MCP Agent API進行交互。

## 可用的腳本

在項目目錄中，您可以執行：

### `npm start`

在開發模式下運行應用程序。
打開 [http://localhost:3000](http://localhost:3000) 在瀏覽器中查看。

### `npm run build`

構建生產版本的應用程序到 `build` 文件夾。

## 功能特點

- 與MCP Agent API進行通信
- 支持對話記憶管理
- 響應式設計
- 支持Markdown和代碼高亮
- 顯示工具使用情況

## 安裝

```bash
# 安裝依賴
npm install

# 開發模式啟動
npm start

# 構建生產版本
npm run build
```
